# Cobbleverse Neo

An Unofficial NeoForge port of the Cobbleverse Fabric modpack for Minecraft 1.21.1.

## Installation

1. Download the latest release zip from [Releases](https://github.com/UmushiUmushi/Cobbleverse-Neo/releases)
2. Open CurseForge and click **Create Custom Profile** → **Import**
3. Select the downloaded zip
4. Launch the profile

## What is this?

Cobbleverse is a Cobblemon-based modpack originally built on Fabric. Cobbleverse Neo is a from-scratch port to NeoForge using Sinytra Connector for Fabric mod compatibility.

See `PORT_DOCUMENTATION.md` inside the modpack for full details on the port process, mod replacements, and known issues.

This is an unofficial NeoForge port of Cobbleverse, originally created by Lumyverse. All credit and ownership belongs to the original creators."
